# 👗 오늘 뭐 입지? OOTD

내 옷장을 기반으로 **날씨 · TPO · 체형**을 고려해  
오늘 가장 잘 어울리는 코디를 추천해주는  
**AI 데일리 스타일링 Streamlit 앱**입니다.

> “옷은 많은데 왜 입을 옷은 없을까?”  
> → 이제 내 옷장으로 해결하세요.

---

## ✨ 주요 기능

### 1. 오늘의 OOTD 추천 (오코추)
- 오늘의 **날씨(자동/수동)** 반영
- **상황(TPO)** 선택 (학교, 직장, 결혼식, 운동 등)
- **체형 정보**(골격, 메모) 입력
- 내 옷장 기반으로 코디 1세트 자동 추천
- 추천 이유 설명 제공

### 2. 똑똑한 옷장 관리
- 의류 **이미지 업로드 / 구매 링크 등록**
- 카테고리, 색상, 기장, 넥라인 관리
- 보온감 / 포멀함 슬라이더
- 이름 기반 **자동 태깅(간단 버전)**
- 필터 & 검색 기능

### 3. 인기 코디 피드
- 추천 코디 저장 및 피드 게시
- 좋아요 기능
- 좋아요 + 최신성 기반 트렌딩 정렬
- 실제 유저 코디를 레퍼런스로 활용 가능

### 4. (선택) OpenAI 기반 AI 추천
- OpenAI API 사용 시
- 규칙 기반 추천보다 더 자연스러운 코디 + 설명 생성

---

## 🛠 기술 스택

- **Frontend / App**: Streamlit
- **Language**: Python 3.9+
- **AI (선택)**: OpenAI API
- **Weather API (선택)**: Open-Meteo
- **Storage**: Local JSON + 이미지 파일
